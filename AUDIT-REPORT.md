## 3. Smogon Damage Calculator との比較

### 完全一致項目 ✅

| 項目 | 状態 |
|------|------|
| pokeRound 関数 | **完全一致** |
| 全MOD定数 | **完全一致**（全20個以上） |
| 修正値適用順序 | **一致** (Spread→Weather→Crit→Random→STAB→Type→Burn→Final) |
| タイプ相性計算 | **一致** |
| STAB値 (1.5x / 2.0x Adaptability) | **一致** |
| 壁修正値 (Singles 0.5x / Doubles 0.667x) | **一致** |
| 天候修正値 | **一致** |
| 急所ランク処理 | **一致** |
| 乱数ループ (85-100, 16段階) | **一致** |
| 基本ダメージ式（Lv50） | **一致** |

### 構造的差異（結果は概ね一致、稀に±1ずれ）

| 差異 | 影響 | 詳細 |
|------|------|------|
| **Final mod チェーン方式** | 低 | Smogon: 全final modを1つにチェーン後適用 / 当ツール: 個別に適用。複数final mod同時使用時に±1 |
| **Atk/Def stat修正の浮動小数点使用** | 低 | Smogon: 4096ベースchainMods / 当ツール: Math.floor(stat * multiplier)。複数修正時に±1 |
| **BP修正の個別適用** | 低 | Smogon: BPもchainMods / 当ツール: 個別applyMod。実使用ケースで差異ほぼなし |
| **OF32/OF16 欠如** | 極低 | 32bit/16bitオーバーフロー保護なし。Lv50環境で実質影響なし |

### 差異の実影響評価

上記の構造的差異は、**複数の修正値が同時に適用される場合**にのみ±1の差が出る可能性があります。例：

- Reflect + Life Orb + Filter が同時に適用 → 最大±1
- Huge Power + Choice Band が同時に適用 → 最大±1

単一修正値の場合（全テストケースの大半）は**完全一致**します。
